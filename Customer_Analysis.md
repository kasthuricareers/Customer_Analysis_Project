# **Customer Analysis using Python, SQL & Power BI**

**Project Overview**

This project demonstrates an end-to-end Data Analytics workflow using a customer dataset. The objective is to perform data cleaning, 
exploratory analysis, SQL-based querying, and dashboard development to generate meaningful business insights.
The project covers the complete analytics process from raw data preparation to interactive reporting.

**Objectives**

•	Understand and explore customer data.

•	Clean and preprocess data using Python.

•	Perform business analysis using SQL queries.

•	Create interactive dashboards and visualizations using Power BI.

•	Generate actionable customer insights for decision-making.
________________________________________
**Tools & Technologies**

•	Python (Pandas, NumPy)

•	SQL (MySQL)

•	Power BI (DAX)
________________________________________
**Project Workflow**

**1. Data Understanding**

•	Examined dataset structure and columns.

•	Identified numerical, categorical, and date fields.

•	Understood customer-related attributes and business context.

**2. Data Cleaning using Python**

Performed data preprocessing tasks such as:

•	Handling missing values

•	Removing duplicate records

•	Correcting data types

•	Renaming columns where necessary

•	Fixing inconsistent or invalid values
________________________________________
**3. Data Analysis using SQL**

Imported the cleaned dataset into MySQL and performed analytical queries:

•	Creating and managing the project database and tables.

•	Analysing customer demographics and purchase behaviour.

•	Comparing revenue generated by different customer segments.

•	Evaluating the impact of discounts on customer spending.

•	Comparing average purchase amounts across shipping methods.

•	Analysing subscriber vs non-subscriber performance.

•	Identifying repeat buyers and subscription trends.

•	Measuring revenue contribution across age groups.

•	Finding high-performing product categories based on sales.

SQL Concepts Used: SELECT, WHERE, GROUP BY, ORDER BY, COUNT(), SUM(), AVG(), HAVING.
________________________________________
**4. Data Visualization using Power BI**

Created an interactive dashboard featuring:

•	KPI Cards

•	Bar Charts

•	Pie Charts

•	Line Charts

•	Tables

•	Slicers & Filters

The dashboard enables users to explore customer trends and key business metrics dynamically.

**KPI Metrics**

•	Total Customers: 3.9K

•	Total Purchase Amount: $233.1K

•	Average Purchase Amount: $59.77

•	Average Review Rating: 3.75

**Visualizations**

•	Payment Method Breakdown

•	Revenue by Product Category

•	Season-wise Revenue by Category

•	Purchase Analysis by Age Group

•	Customer Purchase Summary Table

**Interactive Filters**

•	Subscription Status

•	Gender

•	Category

•	Shipping Type

**Key Insights**

•	Clothing generated the highest revenue among all categories.

•	Young adults contributed the highest purchase amount.

•	Revenue remained consistent across different seasons.

•	Subscribers and non-subscribers can be compared using interactive filters.

•	Multiple payment methods contribute almost equally to overall sales.
